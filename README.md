## 用vue做一个好看的小红书

## 技术栈
- vue全家桶(vue,vue-router,vuex)
- easy-mock造数据
- axios请求数据
- vue-awesome-swiper组件实现图片滑动

## 已实现功能
- 笔记页面及跳转到笔记详情页
- 搜索笔记或者用户
- 关注用户并且在关注页面展示已关注的用户
- 关注取关用户
